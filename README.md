# Exploratory Data Analysis (EDA) of automobile dataset.

Exploratory Data Analysis (EDA) of automobile dataset using descriptive statistics and data visualization techniques in Python. The project focuses on understanding data distributions, relationships between variables, and identifying patterns using plots and statistical measures.

The sprawozdanie_SW.pdf file contains a description and analysis of code in statystykaWizualizacja.ipynb (in Polish).

## Tools
- Python
- pandas
- numpy
- matplotlib
- seaborn
- scipy
- sklearn
